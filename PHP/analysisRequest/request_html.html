Query Request Page

A simple way to request analyses of Iperf network testing data.

Contact Page
Test Type
Please choose which type of tranmission techonology (i.e., the test conducted) you would like to have analyzed.
File Info Criteria
Please choose the basic test file information by which to filter test files you will consider. For instance, if you wish to only consider test files that were created by a device using an AT&T, you would choose the Cellular Network Carrier field, and insert AT&T
Test Type Criteria
Based on your choice in the Test Type field, this area will populate with fields by which to specify which types of individual tests to consider. For instance, if you were to choose TCP in the Test Type field, this area would populate with options such as Average Download Speed, Download Speed Standard Deviation, and many more.
Statistics
Once you have selected your criteria by which to narrow the data that will be used, this option provides a more specific grouping choice.
Submit
Submits the form to be processed so that the query request may be generated.
